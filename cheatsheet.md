### led cheatsheet

Command | Function
------- | -------
e <name> | Open file
x <name> | Close file
f <name> | Rename current file or make new file
a | Add line 
c | Change line 
d | Delete line (commands like 6a/6c/6d also work)
= | Check all the lines
. | Check the current line
<num> | Move to line number <num>
y | yank line
p | paste line (commands like 6y/6p also work)
u | Undo
w | write
q | quit
q! | force quit
wq | write and quit
clear | Clear Screen
